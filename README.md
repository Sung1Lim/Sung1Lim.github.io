## Running Locally (macOS)

Requires **Ruby 3.1** (Jekyll 3.9 is incompatible with Ruby 3.2+).

```bash
# Install Ruby 3.1 if not already installed
brew install ruby@3.1

# Install dependencies
/opt/homebrew/opt/ruby@3.1/bin/bundle install

# Start local server
/opt/homebrew/opt/ruby@3.1/bin/bundle exec jekyll serve --port 4000
```

Then open [http://localhost:4000](http://localhost:4000) in your browser.
